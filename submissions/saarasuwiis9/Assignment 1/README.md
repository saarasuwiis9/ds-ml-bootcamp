Machine Learning Research Assignment
Description

This assignment explains the basic concepts of Machine Learning, including types of learning, overfitting, training and testing data, and real-life applications.

Topics Covered

Definition of Machine Learning

Supervised vs Unsupervised Learning

Overfitting and how to prevent it

Training data and test data

Machine Learning in healthcare

Purpose

The purpose of this assignment is to understand how machine learning works and how it is used in real-life situations.
