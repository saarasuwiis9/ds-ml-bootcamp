# Machine Learning: Research Assignment

This repository contains my introductory research assignment on the fundamentals of Machine Learning (ML). The goal of this project is to explain core ML concepts in a clear, beginner-friendly manner.

## What's Included?
* **Definition of ML:** A simple explanation using real-life examples.
* **Learning Paradigms:** A comparison between Supervised and Unsupervised learning.
* **Model Challenges:** An overview of Overfitting and how to prevent it.
* **Data Strategy:** Explanation of why we split data into Training and Test sets.
* **Case Study:** A brief look at how ML is applied in business and transportation.

You can read the full assignment by opening the `assignment 1.md` file in this repository.
