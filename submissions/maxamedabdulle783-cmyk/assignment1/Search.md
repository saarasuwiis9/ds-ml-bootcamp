1. Machine Learning Definition & Example
Definition: Machine Learning (ML) waa qayb ka mid ah AI-da oo u saamaxda nidaamyada kombuyuutarka inay xogta wax ka bartaan, aqoonsadaan qaababka (patterns), oo ay gaaraan go'aanno.

Core Principle: Nidaamyadu waxay ku tiirsan yihiin xog, algorithms, iyo geedi-socod tababar si ay u sameeyaan saadaalin.

Real-Life Example (Email Filtering):

Learning: Algorithm-ku wuxuu aqoonsadaa ereyada soo noqnoqda sida "free money" ama "urgent action".

Action: Wuxuu kala saaraa fariimaha muhiimka ah iyo kuwa khatarta ah (Spam) isagoo eegaya macluumaadka soo diraha.

2. Supervised vs. Unsupervised Learning
Supervised Learning:

Method: Waxay isticmaashaa xog la sumadeeyey (Labeled Data) si loo baro algorithm-ka map-ka u dhexeeya input-ka iyo output-ka.

Example: Saadaalinta dhibcaha ardayga iyadoo loo eegayo saacadaha uu wax akhriyey (Grade Prediction).

Unsupervised Learning:

Method: Waa nooc ka mid ah ML halkaas oo model-ka lagu tababaro xog aan la sumadayn (Unlabeled Data).

Logic: Waxay u shaqaysaa sida ilmo yar oo kaligii wax sahaminaya si uu u helo qaabab ama kooxaysi qarsoon (Clustering).

3. Overfitting: Causes & Prevention
Common Causes:

Xogta oo leh buuq ama wasakh (Noisy/Dirty Data).

Xogta tababarka oo aad u yar (Limited Training Data).

Adeegsiga sifooyin aan khusayn mawduuca (Irrelevant Features).

Prevention Methods:

Dropout: In si aan kala sooc lahayn loo damiyo qaybo ka mid ah "neurons-ka" xilliga tababarka.

Pruning: In la yareeyo murgaca model-ka si looga dhigo mid fudud.

Data Augmentation: In la kordhiyo lana kala duwo xogta tababarka.

4. Data Splitting (Training vs. Test)
The Process: Xogta waxaa loo qaybiyaa labo qaybood oo kala madax-bannaan: Training Set (oo model-ka lagu baro) iyo Test Set (oo lagu tijaabiyo).

Necessity: * In la xaqiijiyo in model-ku bartay xeerar guud halkii uu ka qaybi lahaa (memorize) tusaalooyinka kaliya.

In laga hortago "Overfitting" iyo "Data Leakage".

In la helo cabir dhab ah oo muujinaya sida model-ku ugu shaqayn doono xog cusub oo dunida dhabta ah ka timid.
